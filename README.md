# Contexia - GPS Financiero para PyMEs

Plataforma de Inteligencia Financiera que ayuda a las PyMEs en Latinoamérica a gestionar su flujo de caja, impuestos y riesgo regulatorio.

## Estructura del Proyecto

- `/` (Raíz): Landing page y portal estático (Vercel).
- `Contexia_Daschboard/`: Dashboard premium construido en React + Vite (Vercel Subdomain).
- `apps/backend/`: Backend API construido con FastAPI (Railway).

## Tecnologías

- **Frontend**: HTML/JS (Landing), React (Dashboard), Tailwind CSS.
- **Backend**: FastAPI (Python), Supabase (PostgreSQL), JWT.
- **Infraestructura**: Vercel, Railway, Supabase.

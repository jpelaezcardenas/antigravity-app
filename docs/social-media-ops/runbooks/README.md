# Docs — Runbooks

Directorio para documentar incidentes recurrentes y sus soluciones.

## Formato obligatorio por runbook

```markdown
# [Nombre del incidente]
- **Fecha de primera ocurrencia:**
- **Causa raíz:**
- **Síntomas:**
- **Pasos de mitigación:**
- **Cómo prevenir recurrencia:**
- **Última actualización:**
```

Per global rules §8: "Nunca fallar dos veces por lo mismo."
